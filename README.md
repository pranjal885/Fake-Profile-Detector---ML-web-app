# 🕵️‍♀️Fake Profile Detection System

Live Demo : https://fake-profile-detector.streamlit.app/

An AI-based system that detects fake online profiles using Machine Learning.

## Tech Stack
- Python
- Streamlit
- Scikit-learn

## Model
- Logistic Regression

## Features
- Confidence-based prediction
- Input validation
- Dataset transparency

Note: Dataset is included for reference and was used during model training. 
The deployed application uses a pre-trained model for inference only.
